#  AI-Based Predictive CPU Scheduler

## Overview
This project implements an intelligent CPU scheduling system that uses Machine Learning to predict process burst times and optimize scheduling decisions.

## Objectives
- Reduce average waiting time
- Improve turnaround time
- Enhance CPU utilization
- Compare with traditional scheduling algorithms

##  Features
- Predict CPU burst time using ML
- Simulate scheduling algorithms (FCFS, SJF)
- Performance comparison
- Visualization of results

# Technologies Used
- Python
- Machine Learning (Linear Regression / Decision Tree)
- Operating System Scheduling Algorithms
